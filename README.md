# Blue Alpha Lib
![logo](./img/blue-alpha-lib.svg)

BTLE communication library for Sony Alpha series cameras, written in Rust.

# Features
## Implemented
* GNSS coordinate and time API

## Planned
* Remote control API
* Camera control information read API
